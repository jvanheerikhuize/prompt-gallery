# Contributing

Thank you for considering a contribution to this template.

## What Lives Here vs. the Governance Repo

| Change | Where to contribute |
| ------ | ------------------- |
| Template structure, workspace layout, artifact templates | This repo |
| A-SDLC controls, stages, directives, schemas | [jvanheerikhuize/a-sdlc](https://github.com/jvanheerikhuize/a-sdlc) |

Do not propose changes to `a-sdlc/` here — it is a read-only submodule.

## How to Contribute

1. **Open an issue first** for anything beyond a trivial fix — describe the problem and proposed solution before writing code.
2. **Fork and branch** — use a descriptive branch name (e.g. `feat/add-stage-7-template`).
3. **Keep changes focused** — one concern per pull request.
4. **Update documentation** — if you change behaviour, update `AGENTS.md`, `README.md`, and `.agent/settings.yaml` to match.
5. **Open a pull request** against `main` — fill in the PR template.

## Reporting Issues

Use [GitHub Issues](../../issues). Include:

- What you expected to happen
- What actually happened
- Steps to reproduce
